 1. 从矩阵视角理解消元 (Elimination Matrices)

我们之前的操作是“第 $i$ 行减去 $l$ 倍的第 $j$ 行”。在矩阵世界里，**对矩阵行的操作 $\iff$ 左乘一个初等矩阵 (Elementary Matrix)**。

- **初等矩阵 $E_{ij}$**：
    
    - 构造方法：将单位矩阵 $I$ 中第 $i$ 行第 $j$ 列的 $0$ 改为 $-l$（即消元乘数 $l_{ij}$ 的相反数）。
        
    - **作用**：$E_{ij}A$ 的结果就是执行了一次“Row $i$ - $l \times$ Row $j$”的操作。
        
- 整个消元过程：
    
    $$(E_{32} E_{31} E_{21}) A = U$$
    
    其中 $U$ 是上三角矩阵 (Upper Triangular Matrix)。
    

## 2. A = LU 分解 (The $A=LU$ Factorization)

与其看着一堆 $E$ 乘在 $A$ 的左边，不如把它们移到等式右边：$A = (E_{21}^{-1} \dots) U$。

- **L (Lower Triangular)**：下三角矩阵。
    
    - 它是所有初等矩阵逆矩阵的乘积：$L = E_{21}^{-1} E_{31}^{-1} E_{32}^{-1}$。
        
    - **神奇性质**：$E$ 的逆矩阵很好求（把 $-l$ 变成 $+l$），而且它们乘在一起时（按正确顺序），**乘数 $l_{ij}$ 会直接落入 $L$ 的对应位置 $(i, j)$ 中，互不干扰**。
        
    - **对角线**：$L$ 的对角线通常全是 1。
        
- **U (Upper Triangular)**：上三角矩阵。
    
    - 消元结束后的结果（即阶梯形矩阵）。
        
    - **主元 (Pivots)**：位于 $U$ 的对角线上。
        

> **结论**：任何非奇异且不需要行交换的矩阵，都可以分解为 $A = LU$。
> 
> - $L$ 记录了**消元的过程**（乘数）。
>     
> - $U$ 记录了**消元的结果**（主元）。
>     

## 3. 为什么要做 A=LU？

解决 $A\mathbf{x} = \mathbf{b}$ 变得非常快，分为两步（利用三角系统）：

1. **前向代入 (Forward Substitution)**：解 $L\mathbf{c} = \mathbf{b}$，求出中间向量 $\mathbf{c}$。
    
    - 因为 $L$ 是下三角，方程从上往下解，非常容易。
        
2. **回代 (Back Substitution)**：解 $U\mathbf{x} = \mathbf{c}$，求出最终解 $\mathbf{x}$。
    
    - 因为 $U$ 是上三角，方程从下往上解。
        

## 4. 行交换与置换矩阵 (Permutations P)

如果消元过程中主元位置是 0，必须进行**行交换**。

- **置换矩阵 $P$**：由单位矩阵 $I$ 交换行得到。
    
- **性质**：
    
    - $PA$：交换 $A$ 的行（左乘）。
        
    - $AP$：交换 $A$ 的列（右乘）。
        
    - **重要性质**：$P^{-1} = P^T$（$P$ 是正交矩阵）。
        

## 5. PA = LU (通用情况)

如果 $A$ 需要行交换才能完成消元，那么 $A=LU$ 就不成立了。正确的形式是：

$$PA = LU$$

- **解释**：先用 $P$ 对 $A$ 的行进行重排（把合适的主元换上去），然后对这个“好”的矩阵 $PA$ 进行标准的 $LU$ 分解。
    
- 这是计算机软件（如 MATLAB, NumPy）求解线性方程组的标准形式。
