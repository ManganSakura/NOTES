## 1. 线性无关 (Linear Independence)

这是衡量一组向量是否有“多余”成分的标准。

- 定义：
    
    如果一组向量 $\mathbf{v}_1, \dots, \mathbf{v}_k$ 的线性组合为零向量的唯一方式是所有系数都为 0（即 $c_1=0, \dots, c_k=0$），那么这组向量是线性无关的。
    
    $$c_1 \mathbf{v}_1 + \dots + c_k \mathbf{v}_k = \mathbf{0} \implies \text{all } c_i = 0$$
    
- 矩阵视角的判定：
    
    将这组向量作为矩阵 $A$ 的列。
    
    - **线性无关** $\iff$ 零空间 $N(A)$ 只有零向量 $\mathbf{0}$ $\iff$ 秩 $r = n$ (列满秩) $\iff$ 没有自由变量。
        
    - **线性相关** $\iff$ $N(A)$ 含有非零向量 $\iff$ 存在自由变量。
        

## 2. 张成空间 (Spanning a Space)

- 定义：
    
    一组向量 $\mathbf{v}_1, \dots, \mathbf{v}_k$ 张成 (Span) 空间 $S$，意味着 $S$ 包含了这组向量的所有线性组合。
    
- **例子**：矩阵 $A$ 的列向量张成了列空间 $C(A)$。
    

## 3. 基 (Basis)

基是描述一个空间的“最佳”向量组——既不缺斤短两，也不画蛇添足。

- **定义**：空间 $S$ 的一组基，是满足以下两个条件的向量序列：
    
    1. 它们是**线性无关**的（没有多余的）。
        
    2. 它们**张成**了整个空间 $S$（足够生成空间里的任何向量）。
        
- **性质**：
    
    - 一个空间可以有无数多组基（比如 $\mathbb{R}^2$ 可以由 $\hat{i}, \hat{j}$ 张成，也可以由旋转后的两个正交向量张成）。
        
    - 但是！**所有基包含的向量个数是唯一的**。
        

## 4. 维数 (Dimension)

- **定义**：空间 $S$ 的**维数 (Dimension)**，等于其**基中向量的个数**。
    
- **符号**：$\dim(S)$。
    
- **重要空间的维数**：
    
    - **列空间 $C(A)$**：$\dim(C(A)) = r$（即矩阵的**秩 Rank**）。
        
        > 这就是为什么“秩”这么重要，它告诉了我们列空间的真实维度。
        
    - **零空间 $N(A)$**：$\dim(N(A)) = n - r$（即**自由变量的个数**）。
        
    - $\mathbb{R}^n$：维数为 $n$。
        

## 5. 秩-零化度定理 (Rank-Nullity Theorem)

这是线性代数基本定理的第一部分，连接了目标空间和源空间：

$$r + (n - r) = n$$

$$\text{列空间的维数} + \text{零空间的维数} = \text{列的总数 (变量数)}$$

---

**小结**：

- **线性无关** = $N(A) = \{\mathbf{0}\}$。
    
- **张成** = $C(A)$ 填满了空间。
    
- **基** = 线性无关 + 张成。
    
- **维数** = 基里的向量数。
