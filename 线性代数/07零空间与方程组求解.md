## 1. 零空间 (Null Space, $N(A)$)

- **定义**：包含了所有使得 $A\mathbf{x} = \mathbf{0}$ 的解向量 $\mathbf{x}$ 的集合。
    
- **空间位置**：
    
    - $N(A)$ 是 $\mathbb{R}^n$ 的子空间（注意：它在**定义域**里，而列空间 $C(A)$ 在**值域** $\mathbb{R}^m$ 里）。
        
    - 它一定包含零向量 $\mathbf{0}$。
        
- **几何意义**：$N(A)$ 里的向量与 $A$ 的每一行都**垂直**（正交）。
    

## 2. 求解 $A\mathbf{x} = \mathbf{0}$ (Pivot vs. Free)

算法的核心目标是找到所有的“特解”。

### 2.1 算法流程

1. **消元**：将 $A$ 化为阶梯形矩阵 $U$，甚至进一步化为 **行最简形 (RREF, Reduced Row Echelon Form, 记为 $R$)**。
    
2. **区分变量**：
    
    - **主元变量 (Pivot Variables)**：对应列中有主元的变量。
        
    - **自由变量 (Free Variables)**：对应列中**没有**主元的变量。
        
3. **找特解 (Special Solutions)**：
    
    - 如果你有 $n$ 个变量，秩为 $r$，那么你有 $n-r$ 个自由变量。
        
    - **操作**：每次将**一个**自由变量设为 1，其他的自由变量设为 0，然后求出主元变量的值。
        
    - 这样得到的解就是特解。
        
4. **通解**：
    
    $$\mathbf{x} = c_1 \mathbf{s}_1 + c_2 \mathbf{s}_2 + \dots + c_{n-r} \mathbf{s}_{n-r}$$
    
    即零空间是由这 $n-r$ 个特解张成的。
    

## 3. 秩 (Rank, $r$)

- **定义**：矩阵中**主元 (Pivots) 的个数**。
    
- **物理意义**：代表了矩阵包含的“真实信息量”。
    
- **满秩的情况**：
    
    - **列满秩 ($r=n$)**：没有自由变量 $\rightarrow N(A) = \{\mathbf{0}\}$ $\rightarrow A\mathbf{x}=\mathbf{0}$ 只有零解。
        
    - **行满秩 ($r=m$)**：对任意 $\mathbf{b}$ 都有解（列空间填满 $\mathbb{R}^m$）。
        

## 4. 求解 $A\mathbf{x} = \mathbf{b}$ (The Complete Solution)

当等式右边不再是 0 时，解的结构发生了变化。

### 4.1 可解性条件 (Solvability Condition)

并不是所有的 $\mathbf{b}$ 都有解。

- **条件**：$\mathbf{b}$ 必须属于列空间 $C(A)$。
    
- **判别法**：在增广矩阵 $[A | \mathbf{b}]$ 消元后，如果出现“零行 = 非零数” ($0 = d$)，则无解。
    

### 4.2 通解结构 (极重要)

$$\mathbf{x}_{\text{complete}} = \mathbf{x}_p + \mathbf{x}_n$$

- **$\mathbf{x}_p$ (特解/特定解, Particular Solution)**：
    
    - 满足 $A\mathbf{x}_p = \mathbf{b}$ 的任意一个解。
        
    - **求法**：通常将所有**自由变量设为 0**，解出主元变量。
        
- **$\mathbf{x}_n$ (零空间解/齐次解, Homogeneous Solution)**：
    
    - 满足 $A\mathbf{x}_n = \mathbf{0}$ 的所有解（即上面第 2 节求出的线性组合）。
        

### 4.3 几何图景

- $N(A)$ 是过原点的一个子空间（比如通过原点的平面）。
    
- 解集 $\mathbf{x}_p + \mathbf{x}_n$ 是这个子空间进行了**平移**（Shifted）。
    
    - 它不再穿过原点（除非 $\mathbf{b}=\mathbf{0}$）。
        
    - **因此，$A\mathbf{x}=\mathbf{b}$ 的解集通常不是子空间！**（这经常是考点）。
        

---

**小结**：

- $A\mathbf{x}=\mathbf{0}$ 找子空间（特解的组合）。
    
- $A\mathbf{x}=\mathbf{b}$ 找平移的“平面”（一个特解 + 零空间）。
